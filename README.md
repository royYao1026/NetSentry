# NetSentry

## IoT Network Flow Anomaly Detection System

NetSentry is an end-to-end machine learning based cybersecurity monitoring system designed to detect anomalous behaviors in IoT network traffic.

The project simulates a real-world network security monitoring pipeline:

Network Traffic
→ Feature Extraction
→ Machine Learning Detection
→ Real-time Inference
→ Data Drift Monitoring
→ Alert Dashboard


## Objectives

- Detect abnormal IoT network communication patterns
- Build scalable ML inference pipeline
- Monitor model performance under changing traffic distributions
- Deploy a lightweight cybersecurity monitoring system

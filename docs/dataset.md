# Dataset

## CICIoT2023

Source:
https://www.unb.ca/cic/datasets/iotdataset-2023.html


## Description

CICIoT2023 contains network traffic generated from IoT devices under normal and attack scenarios.

The dataset provides network flow features extracted from packet-level traffic.


## Objective

The goal of NetSentry is to classify network flows as:

- Normal traffic
- Malicious traffic


## Attack Categories

- DDoS
- DoS
- Reconnaissance
- Brute Force
- Web Attacks
- Malware


## Features

Examples:

- Flow duration
- Packet length
- Number of packets
- Source/Destination information
- Protocol information
- Traffic statistics


## ML Task

Primary Task:

Binary Classification
